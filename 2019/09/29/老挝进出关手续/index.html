二、签证

签证是你想去的国家给你护照上面签注、盖印、附文（如另纸签证），以示允许你出入其国境的手续。

（一）落地签

申请签证一般有两种途径：
1.在昆明老挝驻中国领事馆申请办理；
2.本人持护照原件在抵达老挝后入关处申请办理（就叫落地签）。

需要材料：

  1. 2寸照片1张（不限底色，贴在落地签申请表上）
  2. 20美元（没有美元可以支付人民币200元）
  3. 邀请函

    邀请函由老挝区域部办理，需要提供护照首页清晰照片，约需时间为5个工作日。办好以后发扫描件给你。
    用法：

    1. 将邀请函扫描件打印出来，1份（彩色）。

    2. 带在身上，到老挝入关处办理落地签和其他资料一起交给工作人员（入境卡、落地签申请表、邀请函、20美元）

    3. 办好后检查是否有错（T-B2是商务签，对的。T-B3是旅游签，错的，要求改正）。

    4. 索要发票

  4. 入境卡（飞机上会发放，入关处也可以领取）
  1. 落地签申请表（入关出领取，现场填写）
